SETUP REQUIRED (3 STEPS):
--------------------------

1. Deploy to your org:
   sf project deploy start --target-org YOUR_ORG

2. Schedule the nightly job (in Developer Console):
   GitHubSyncScheduledJob.scheduleNightlySync();

3. Add button to Sync Jobs list view:
   Setup → Object Manager → Sync Job → Buttons, Links & Actions
   Add "Sync All Repositories" to list view button layout
